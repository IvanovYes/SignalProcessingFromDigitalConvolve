# SignalProcessingFromDigitalConvolve

эмулировать измерения сигнала и проводить его вейвлет-преобразование при помощи готовых реализованных математических операций. Основной такой операцией применительно к вейвлет-образованию является операция дискретной свертки, которая реализуется функцией convolve в режиме «valid» из библиотеки NumPy. При этом для корректности работы данной операции свертки необходимо задавать массивы сигнала и базисной функции как M = length и N = 2*length-1, так как в режиме «valid» функция convolve вернет массив размерностью max(M,N) – min(M,N) + 1, что как раз будет равно исходной длине сигнала length. Это также полностью соотносится с правилом для задания пределов интегрирования непрерывного вейвлет преобразования, как в 2 раза большего чем длина сигнала.
